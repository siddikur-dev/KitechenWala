## 🚀 Features

- 📋 Browse all orders from **orders.json**
- 🔥 Send orders to **Cooking section**
- ✅ Move cooked items to **Ready Serve list**
- 🧹 Auto-remove from previous list when moved
- 🎨 Styled with **Tailwind CSS**
- ⚛️ Built using **React 18**
